Guide

実際にホームページにする手順

1. GitHub アカウントを作る / ログインする

まだなら GitHub のアカウントを作ります。すでにあるならそのままで大丈夫です。

2. 公開リポジトリを作る

literary-homepage のような名前で Public リポジトリを1つ作ります。

3. このHTMLを index.html で置く

このコード全体を index.html という名前で保存し、GitHub にアップロードします。

4. GitHub Pages を有効化する

リポジトリの Settings → Pages を開き、Source を Deploy from a branch、Branch を main / /root に設定します。

5. 公開URLを確認する

数分すると https://ユーザー名.github.io/リポジトリ名/ で見られるようになります。

6. 記事を増やす

まずはトップページだけで十分です。その後、個別記事ページや記事一覧ページを足していけば足ります。
